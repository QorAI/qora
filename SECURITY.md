# Security Policy

## Supported Versions

Any version of Qora used in the current stable release version or above of QorAI Browser, or in any other repo under https://github.com/qorai. This includes [qorai-core](https://github.com/qorai/qorai-core) and [qorai-ios](https://github.com/qorai/qorai-ios).

## Reporting a Vulnerability

See https://hackerone.com/qorai for details.
